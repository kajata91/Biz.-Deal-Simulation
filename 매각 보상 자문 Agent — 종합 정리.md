# 매각 보상 자문 Agent — 종합 정리

> ① **기본·원칙**, ② **일하는 방식(프로세스)**, ③ **실제 사례 기반 시뮬레이션** 을 제공하는 자문 도구.

## 2. 화면 구성 (v5.0 · 7개 메뉴)

상단 가로 네비게이션. 활성 메뉴는 다크 pill.

```
대시보드 │ 원칙 │ 프로세스 │ 시뮬레이션 │ 사례 │ 지식 │ 자료
```

| 메뉴 | 내용 |
|---|---|
| **대시보드** | 핵심 KPI(36 PEF·28 SK사례·10 프리셋) + Credo 7 + 위로금 결정 4대 변수 + 빠른 이동 |
| **원칙** | 리더 7대 신조(Credo) + 실무 11대 원칙 + 보상 벤치마크 수치 |
| **프로세스** | 딜 생애주기 **8단계 Playbook**(아래 4절). 진단·산정·협상 매트릭스 내장 |
| **시뮬레이션** ★ | 협상 전략·조건·금액 **매트릭스(30변수)** + 위로금·충당금 산정. 실제·진행 딜 프리셋 10종 |
| **사례** | 중요 사례(글로벌5+한국6) / 🏦 PE 매수자 대응(36개 PEF) — 서브탭 |
| **지식** | 석학 이론 / Risk Point / 전략적 토대 / 실무 데이터·템플릿 — 서브탭 |
| **자료** | 자료 다운로드(지식문서 내장) / 참고 출처 — 서브탭 |

> 사용자가 정의한 목적 **"기본·원칙 + 프로세스 + 사례 기반 시뮬레이션"** 3축이 그대로 메뉴가 됨.

## 3. 핵심 원칙 (Credo 7 · 요약)

1. **존엄 우선** — 사업을 파는 것이지 사람을 버리는 게 아니다.
2. **남는 자가 지켜본다** — 떠나는 직원 처우가 잔류 조직 신뢰·몰입을 결정.
3. **공정성은 과정에서** — 절차·상호작용 공정성 > 금액.
4. **나쁜 소식은 빠르게·직접·일관되게.**
5. **흑자매각일수록 두텁게** — 박탈감 반영.
6. **기준은 사전에, 예외는 투명하게.**
7. **약속은 끝까지** — Day-1 이후 12~24개월까지 이행.

## 5. 핵심 도구 — 협상 전략 매트릭스

**30개 변수**의 강도/수준을 선택(또는 사례 프리셋) → 사람 리스크(R)·매도자 레버리지(L)를 점수화 → **협상 포지션 + 우선 관철 조건(필수/권장) + 위로금 권장 밴드(개월)** 산출.

- **변수 6영역**
  - A.딜 특성: 규모 · 구조(주식/영업/자산양수도) · 지분율 · 사유 · 시급성
  - B.재무: 수익성 · 부채 · 성장성 · 차익(보상 여력)
  - C.Buyer: PE 아키타입(A~E) · 보유전략 · 레버리지 · 국적 · 트랙레코드
  - D.임직원: **고용승계 구조** · 핵심인력 의존 · 근속·연령 · 인력규모 · 박탈감 · **해외이전**
  - E.노조: 존재 유무 · 강도 · 상급단체 · 고용안정협약 · 분규이력
  - F.매도자(SK): **SK 브랜드 라이선스** · 평판·여론 · LP·규제 노출 · 잔류영향 · 정부관심
- **포지션 4사분면**: 강경 방어 / 핵심조항 우선+레버리지 보강 / 고용 우호+성장 인센티브 / 표준+절차 공정성
- **위로금 로직**: 고용승계 구조 씬베이스 × PE리스크 × 근속 × 박탈감 × 우량, 해외이전 가산, 차익 affordability 밴드(상한 36개월)

## 6. 🔒 SK 실적 캘리브레이션 (내부 전용 — 공유 금지)

> SK 「매각위로금 사례」 28건(2021~2027)으로 매트릭스 위로금 계수를 보정. 원자료·상세는 프로젝트의 `sk-calibration.md`(로컬 전용)에만 보관.

### 가장 중요한 발견 — 위로금은 ‘고용승계 구조’가 1차 결정 (매각 대금과 무관)

| 티어 | 고용승계 구조 | SK 실적(1인 평균) |
|---|---|---|
| ① 잔류·고용승계(이동 없음) | 고용 그대로 | 미지급~소액 |
| ② 지분매각·단순 이동 | 근로관계 자동 유지 | 500~1,500만 |
| ③ 사업·영업양도 | 사업부 포괄 양도 | 1,600~5,000만 |
| ④ **물적분할·신설법인 전적(carve-out)** | 신설법인 전적 | **6,000~1억700만(본사+신설 분할)** |
| ⑤ 해외 이전 | 해외법인 전적 | 1억+(정착·주재 1년) |

### 진행 중 딜 매트릭스 역산 (요약)

| 딜 | 매트릭스 역산 | 비교 |
|---|---|---|
| **원스토어→넥써스** | ~2.3~2.8개월(≈1,200~1,500만) | **회사 제안 1,200만과 정합**. 노조 2~3,000만은 ‘영업양도’ 프레임 → 구조 해석 충돌. 타결선 ~1,500~2,000만 |
| **SK E&S 재생→KKR** | ~16~20개월(≈9,500만~1.2억) | SK 추정 1.1억 정합(+월급300% 리텐션) |
| **SK E&S 도시가스(미정)** | 권장 ~9~11개월(≈5,300~6,800만) | 영업양도·규제 인프라 |
| **SK실트론→두산** | 구조 ~1,800만 | 추정 5,000만 − 1,800 ≈ **임단협 격려금 + 국가핵심기술 리텐션** |

> 핵심: 구조상 ‘이동’(소액)인데 실제가 크면, 그 차이는 위로금이 아니라 **격려금·리텐션 레이어**다.

## 7. 위로금 결정 원리 (이 Agent의 결론)

1. **위로금 ≠ 매각 대금.** 매각 대금 규모로 위로금을 가늠하는 건 오류.
2. **1차 결정요인 = 고용승계 구조** (이동=소액 / 신설법인 전적=두텁게 / 정리=별도 법정요건).
3. **부가 요인 = 인수사 아키타입 · 해외이전 · 노조.**
4. **“얼마”보다 “어떻게”** — 절차·상호작용 공정성이 분쟁을 가른다.
5. **경영진(MIP·리텐션)과 일반직원은 분리 설계.**
6. **PE 매수자는 공개 메시지를 액면 신뢰 금물** — 고용보장·위로금은 SPA 명문화로만 확보, 재매각 대비 flow-down 필수.

1. `index.html`을 **실제 브라우저**로 연다.
